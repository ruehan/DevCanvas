- 날짜: 2026-07-09
- 브랜치: feat/edit-patch
- 최종 판정: 통과 (1라운드, 지적 0)

## 배경
전체 재생성(ADR-0018)이 실 GLM 에서 timeout/429(출력 과대). 편집을 부분 패치로 전환:
LLM 이 변경 최상위 섹션만 반환(GenerationResultPatch) → merge_patch 로 교체 병합.

## 1라운드 (code-reviewer, api 스코프)
- 판정: **통과 (지적 0)**
- 검증: verify-all.sh EXIT 0 — api(ruff/mypy strict/pytest, 테스트 추가/갱신), web(변경 없음).
- 확인(리뷰어): merge_patch 의 model_dump→update→model_validate 라운드트립이 중첩 타입 보존,
  exclude_none 이 None 섹션만 생략(빈 리스트 `[]` 는 보존), 부분 nested 패치 시 기본값 자동 충전,
  미제공 섹션 원본 유지.
